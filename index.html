WinUIEx

A set of extension methods and classes to fill some gaps in WinUI 3, mostly around windowing and unit testing.

NuGet Package:

  • WinUIEX

Features

  • Window Extension methods
  • HWND Window Handle Extensions methods
  • Extended WindowEx Window class
  • Window Manager
  • Splash screen
  • OAuth Web Authenticator
  • Custom Backdrops
  • Tray Icon support including WinUI 3 based context menus.
  • NumberBoxInt32, NumberBoxDecimal and NumberBox<T>
  • TitleBar control (Deprecated in favor of WinAppSDK 1.7 TitleBar)

And more to come...
